# Footer de la UTSJR

Este es un pequeño proyecto que alberga el concepto de la implementación del nuevo footer del SIISC de la UTSJR.
Se optó por utilizar una funcionalidad experimental de CSS (la regla `@property`) para añadir efectos visuales
al diseño, sin embargo, debido a su baja asimilación al momento de realizar este proyecto, ésta seguramente
será eliminada de la versión final del footer.
